
## 1. 你的错误代码

```c++
for(auto it = mp.rbegin();it != mp.rend();it--)  // ❌ 错误
```

## 2. 核心错误：**反向迭代器 不能 --，只能 ++**

- `rbegin()` = **反向开头**（指向 map 最后一个元素）
- `rend()` = **反向结尾**（指向 map 第一个元素的前面）
- **反向迭代器往前走，用 ++，不是 --**

你写 `it--` 等于**往回倒着走**，永远到不了 `rend()`，直接**无限循环 + 乱输出**！

---

## 3. 正确写法（必须 ++）

```c++
for(auto it = mp.rbegin(); it != mp.rend(); it++)  // ✅ 正确
    cout << it->first << " ";
```

---

## 4. 一句话口诀（背这个）

- **正向迭代器（begin/end）**：往前走用 `++`
- **反向迭代器（rbegin/rend）**：往前走也用 `++`

**反向迭代器永远 ++，绝对不能 --！**

---

## 5. 完整正确示例


```c++
map<int,int> mp = {{1,2},{3,4},{5,6}};

// 从大到小输出 5 3 1
for(auto it = mp.rbegin(); it != mp.rend(); it++){
    cout << it->first << " ";
}
```

`cout:`
```c++
5 3 1
```

---

## 总结（超级重要）


```c++
it++   ✅ 反向迭代器正确写法
it--   ❌ 直接死循环、乱码、崩溃
```
