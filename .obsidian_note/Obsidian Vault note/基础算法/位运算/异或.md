1.异或与格雷码
https://leetcode.cn/problems/gray-code/description/
 二进制码转换成格雷码公式:i ^ (i >> 1);

2.原二进制的第 n 位 ^ 原二进制的第 n+1 位 = 格雷码的第 n 位；
原二进制的第 n 位 = 原二进制的第 n+1 位 ^ 格雷码的第 n 位

普及异或运算：
异或运算：
自反性：x ^ x = 0（相同数异或结果为 0）；
归零性：x ^ 0 = x（任何数和 0 异或等于自身）；
交换性：a ^ b = b ^ a；
结合律：(a ^ b) ^ c = a ^ (b ^ c)。

证明：
已知 a ^ b = c，
两边同时异或 b：(a ^ b) ^ b = c ^ b，
根据结合律：a ^ (b ^ b) = c ^ b，
根据自反性：a ^ 0 = c ^ b，
根据归零性：a = c ^ b，
再根据交换律：a = b ^ c。
结论：a ^ b = c ⇨ a = b ^ c，反过来 a = b ^ c ⇨ a ^ b = c 也成立，完全等价。

## 问题:
在不知道原码的情况下我格雷码的十进制数怎么求原码对应的十进制数：
```c++
int grayToBinaryDec(int gray_dec) {
    int binary_dec = gray_dec; // 初始化：最高位直接赋值
    // 循环右移，逐位推导
    while (gray_dec > 0) {
        gray_dec >>= 1; // 右移1
        binary_dec ^= gray_dec; // 异或更新原码
    }
    return binary_dec;
}
```
