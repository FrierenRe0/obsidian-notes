# 2.前缀和+hash

## 一、核心概念与整体思路

**前缀和 + Hash 是算法竞赛中处理区间子数组 / 子串问题的经典优化技巧，核心目标是将原本需要 O(n 2 )复杂度的暴力枚举，通过前缀和的转化和哈希表的记录，降到 O(`n`)或 O(`nlogn`)，尤其适合解决 “满足某类条件的区间计数、区间存在性判断” 类问题。**

---
## 二、基础原理：前缀和的转化

我们先回顾前缀和的核心公式，再看它如何和区间条件结合：

设数组 a 的前缀和数组为 s，其中 s[0]=0, s[i]=a[1]+a[2]+⋯+a[i]，则区间 [l,r] 的和为：

**sum(l,r)=s[r]−s[l−1]

**很多题目会把 “区间和满足某条件” 转化为关于 s[r] 和 s[l−1] 的等式 / 不等式，再用哈希表记录 s[l−1] 的信息，在枚举 r 时快速匹配。**

---

## 三、「枚举右，维护左」的完整流程

- **枚举右**：遍历区间的右端点 r，将问题转化为「固定右端点时，有多少个左端点 l 能满足条件」。
- **维护左**：用哈希表记录所有前缀和（或其变形）的出现次数 / 位置，快速查询满足条件的左端点数量，避免重复计算。

例题示范：
以最经典的「和为 k 的子数组个数」为例，拆解完整步骤：

1. **初始化**：

    - 前缀和变量 `pre_sum = 0`（动态维护，不用开数组）
    - 哈希表 `hash_map`，初始存入 `{0: 1}`（对应 s[0]=0，表示前缀和为 0 出现过 1 次）
    - 结果计数 `ans = 0`

2. **枚举右端点 r（遍历数组）**：

    - 更新当前前缀和：`pre_sum += a[r]`
    - 计算目标前缀和：`target = pre_sum - k`（由 s[r]−s[l−1]=k 变形得到）
    - 查询哈希表：如果 `target` 存在，就把它的出现次数加到 `ans` 中（表示有这么多 l−1 满足条件）
    - 将当前 `pre_sum` 存入哈希表：`hash_map[pre_sum]++`

3. **遍历结束，返回 `ans`**

这个流程完美体现了「枚举右，维护左」：

- 每一步只枚举一个右端点 r
- 哈希表中存的是所有 l−1 对应的前缀和信息，是 “维护左” 的核心

---

**四、高频应用场景与变形技巧 (对于前缀和+hash的题型概括)**

### 场景 1：前缀和 + 普通哈希（和为定值的子数组）

- 适用题目：和为 k 的子数组个数、和能被 k 整除的子数组个数
- 核心变形：对于 “区间和能被 k 整除”，条件转化为 s[r]≡s[l−1](modk)，哈希表记录的是前缀和模 k 的余数。

### 场景 2：前缀和 + 有序哈希 / 双指针（前缀和单调）
（**但是滑动窗口更方便）
- 适用题目：全正数数组中，和大于 / 小于等于 k 的子数组个数
- 关键前提：数组元素全为正 → 前缀和数组严格单调递增，此时可以用有序哈希（如 C++ 中的 `std::map`）或双指针，在 O(`nlogn`) 内完成查询。

### 场景 3：前缀和 + 状态哈希（多条件约束）

- 适用题目：含 0/1 的子数组中，0 和 1 个数相等、最长子数组长度
- 核心变形：将 0 转化为 - 1，此时 “0 和 1 个数相等” 等价于区间和为 0，再用前缀和 + 哈希记录前缀和第一次出现的位置，计算最大长度。

# 4.二维前缀和
![配图](../../../截图文件/Pasted%20image%2020260424194154.png)![配图](../../../截图文件/Pasted%20image%2020260424195141.png)

```c++
int n,m;
cin >> n >> m;
//输入grid矩阵
vector<vector<int>> grid(n,vector<int>(m));
for(int i = 0;i < n;i++) {
	for(int j = 0;j < m;j++) {
		cin >> grid[i][j];
	}
}
//建立二维矩阵前缀和
vector<vector<int>> sum(n + 1,vector<int>(m + 1));
for(int i = 0;i < n;i++) {
	for(int j = 0;j < m;j++) {
		sum[i + 1][j + 1] = sum[i + 1][j] + sum[i][j + 1] - sum[i][j] + grid[i][j];
	}
}

//解决对于任意矩阵的元素和
int i1,j1,i2,j2;// based 0-i,j
cin >> i1 >> j1 >> i2 >> j2;
//加1转以based - 1为底的矩阵下标
int ans = sum[i2 + 1][j2 + 1] - sum[i2 + 1][j1] - sum[i1][j2 + 1] + sum[i1][j1];
```
