**差分与前缀和的关系，类似**导数**与**积分**的关系。
在做差分数组时，就类似于是在作导数；
在作前缀和时，就类似于是在作积分；

**对于差分的理解可以参考[1094. 拼车 - 力扣（LeetCode）](https://leetcode.cn/problems/car-pooling/solutions/2550264/suan-fa-xiao-ke-tang-chai-fen-shu-zu-fu-9d4ra/)**

# 思考：
- 想要理解差分，其实很简单
- 对于一个区间要增加一个相同的数，对于区间中的所有数字，其中相邻的数中的差没用发生改变，只用在区间两个端点上的数和他们区间外两侧的数的差发生改变；
- 这是我们只要改变这两个位置上的差就行了；

# 差分数组的性质
- 性质 1：从左到右累加 d 中的元素，可以得到数组 a。
- 性质 2：如下两个操作是等价的。
	- 把 a 的子数组 a[i],a[i+1],…,a[j] 都加上 x。
	- 把 d[i] 增加 x，把 d[j+1] 减少 x。

# 步骤：
- 建立差分数组；
	![配图](../../../截图文件/Pasted%20image%2020260519121152.png)

- 利用性质 2，我们只需要 O(1) 的时间就可以完成对 a 的子数组的操作。最后利用性质 1 从差分数组复原出数组 a。

- 注：也可以这样理解，d[i] 表示把下标 ≥i 的数都加上 d[i]。（个人更偏向这种想法）
  ## 这种想法步骤：
	- 建立差分数组，但是差分数组中的d[i]中的值表示的是下标为i的元素在完成所有增加操作的标记，在sum（初始为0）累加到下标为i时，该元素的增减量；


# 二维差分
![配图](../../../截图文件/Pasted%20image%2020260526122051.png)

**本质和一维差分相同，差别就在于是再二维上的操作；

```c++
solve() {
	int n,m;
	cin >> n >> m;
	vector<int> d(n + 2,vector<int>(m + 2));
	int k;
	cin >> k;
	//k次更新
	while(k--) {
		int i1,j1,i2,j2;//(i1,j1)左上角，(i2,j2)右上角 based -> 0
		cin >> i1 >> j1 >> i2 >> j2;
		d[i1 + 1][j1 + 1]++;
		d[i1 + 1][j2 + 2]--;
		d[i2 + 2][j1 + 1]--;
		d[i2 + 2][j2 + 2]++;
	}
	for(int i = 0;i < n;i++) {
		for(int j = 0;j < m;j++) {
			d[i + 1][j + 1] += d[i + 1][j] + d[i][j + 1] - d[i][j];
		}
	}
	//在对二维差分作完前缀和后，d[i][j](based -> 1)为a[i - 1][j - 1]的增加值
}
```
