```c++
bool isprime(int x) {
	for(int i = 2;i * i <= x;i++) {
		if(x % i == 0) return false;
	}
	//这里因为只有x >= 4的所有数字得到筛查。1，2，3这里3这三个数中，也只有1不是质数，则这里判断 x >= 2 即可；
	return x >= 2; 
}
```

## 疑问：

**1.为啥这里如果存在一个（0，x ^ 1/2】的情况就表示是合数？

**思考如果对于一个数来说，他的质因数数量一定是 >= 2,对于合数来说，其质因数数量 > 2,这里一定存在一质因数对 a  *  b == x（定：a <= b && a != 1）;

**我们假设 a ≤ b 
那么一定有：
a * b ≥ a * a;
也就是：
x ≥ a ^ 2
移项得：
x ^ 1/2 ≥ a;

**得：
则我们只要看是否存在一个（0，x ^ 1/2】的情况，有则为合数，否则为质数；
