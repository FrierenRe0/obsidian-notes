# 异或空间线性基

**介绍(关键）：在一堆数据中任意取，所能得到的非0异或和的所有结果，可以被元素个数尽可能少的集合不多不少的等效获取
那么我就说，这个等效集合就是这堆数据的异或空间异或基

**--> 这里注意哦，我在数据（insert函数）插入的顺序不同时，所构建的异或空间异或基可能时不同的，但是这里依旧等价，没有任何问题；


## 如下几个结论是构建异或空间线性基的基础：

- 一堆数字中，任意的a和b，用\( a ^ b \)的结果替代a、b中的一个数字，不会影响异或和的组成

- 一堆数字中，任意的a和b，如果有\( a ^ b = 0 \)，那么舍弃掉a、b其中一个数字，不会影响异或和的组成

- 一堆数字能否异或出0，在求出异或空间线性基之后，需要被单独标记



## 举例说明：现在我用一个例子，来完成对于线性基构建过程便于快速理解
### (普通消元  -->  可以求出关于如下信息：线性基大小，异或和个数，异或和是否有没有0，(子集)最大异或和....等信息)

假设你手里有一个数字集合 $S = \{3, 5, 6\}$。

我们先把它们转换成二进制来看看：

- $3 = (011)_2$
    
- $5 = (101)_2$
    
- $6 = (110)_2$


这个集合通过不选，选1个，选2个，选3个。总共可以“生产”出哪些数字呢？

- 选单个数字：$3, 5, 6$
    
- 两两异或：$3 \oplus 5 = 6$, $3 \oplus 6 = 5$, $5 \oplus 6 = 3$ （发现了吗？它们内部异或居然又绕回去了，这里你因该可以想到我刚刚上面说的基础2了，说明这里存在a ^ b = 0，会有数据的舍弃，继续看后续理解）
    
- 三个异或  或者  不选：$3 \oplus 5 \oplus 6 = 0$


所以，集合 $S$ 能够异或组合出来的所有可能结果是：**$\{0, 3, 5, 6\}$**。

### 思考：我们真的需要全部这 3 个数吗？

其实不需要。如果我们把 $6$ 扔掉，只留下 $\{3, 5\}$：

- 只选 $3$ 或 $5$，可以得到 $3, 5$
    
- $3 \oplus 5$，可以得到 $6$
    
- 都不选（或者自己异或自己），可以得到 $0$


你会发现，仅仅靠 **$\{3, 5\}$**，就能完美生成出原集合所有的异或结果 $\{0, 3, 5, 6\}$！此时，$6$ 就是一个“多余”的元素。

这里的 **$\{3, 5\}$ 就是原集合 $S$ 的一个线性基**。

## 🛠 线性基是怎么运作的？

在实际算法中，线性基通常表现为一个数组（或者说时对应的一个bit位），**每个位置对应二进制的一个最高位（最高位的 1）**。

对于 32 位整数，我们会准备一个大小为 32 的数组 `p`。`p[i]` 存放的数字，其二进制的最高位必须是第 `i` 位。

我们尝试把 $S = \{3, 5, 6\}$ 逐个**插入**到线性基中：

### 1. 插入 $3 = (011)_2$

- 最高位的 `1` 在第 1 位（从右往左，从 0 开始计数：第 0 位是 1，第 1 位是 1）。
    
- 检查 `p[1]`，目前是空的。
    
- 直接把 $3$ 放入 `p[1]`。此时线性基为：`p[1] = 3`。
    

### 2. 插入 $5 = (101)_2$

- 最高位的 `1` 在第 2 位。
    
- 检查 `p[2]`，目前是空的。
    
- 直接把 $5$ 放入 `p[2]`。此时线性基为：`p[1] = 3`, `p[2] = 5`。
    

### 3. 插入 $6 = (110)_2$

- 最高位的 `1` 在第 2 位。
    
- 检查 `p[2]`，发现里面**已经有数字 5 了**！
    
- 为了消除当前位的 `1`，我们让 $6$ 异或 `p[2]`：
    
    $$6 \oplus 5 = (110)_2 \oplus (101)_2 = (011)_2 = 3$$
    
- 剩下的值变成了 $3$。继续看当前 $3$ 的最高位，在第 1 位。
    
- 检查 `p[1]`，发现里面**也已经有数字 3 了**！
    
- 继续消除：$3 \oplus \text{p}[1] = 3 \oplus 3 = 0$。
    
- 数字变成了 $0$，说明 $6$ 彻底被压榨干净了，无法为线性基贡献新的独立维度（它能被现有的线性基组合出来）。**拒绝插入**。
    

最终得到的线性基就是 `p[1]=3`, `p[2]=5`。
