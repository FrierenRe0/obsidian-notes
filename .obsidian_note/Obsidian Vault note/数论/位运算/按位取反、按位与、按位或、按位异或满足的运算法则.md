 四种位运算及其运算律 设操作数为整数 $a,b,c$ 符号说明：
 
 - $\sim$：按位取反（一元运算）
 - $\&$：按位与
 - $\mid$：按位或
 - $\hat{}$（代码写作 `^`）：按位异或

---
 
 一、按位取反 $\sim x$ 一元运算，不存在交换律、结合律，满足以下规律：1. 双重否定律 $$\sim(\sim a) = a$$2. 德摩根律（配合按位与、按位或） $$\sim(a\&b) = (\sim a)\mid(\sim b),\quad \sim(a\mid b) = (\sim a)\&(\sim b)$$ 3. 补码恒等式（计算机有符号整数特有） $$\sim a = -a-1$$
-----------------------------------------------------------------------------------------------------

二、按位与 $\&$（二元运算）
#### 1. 交换律 $$a\&b = b\&a$$ 2. 结合律 $$(a\&b)\&c = a\&(b\&c)$$ 3. 幂等律 $$a\&a = a$$ 4. 吸收律（搭配按位或） $$a\&(a\mid b) = a$$ 5. 分配律（对按位或分配） $$a\&(b\mid c) = (a\&b)\mid(a\&c)$$ 6. 零元 / 单位元性质 $$a\&0 = 0,\quad a\&\text{全1掩码}=a$$ 7. 互补律（搭配按位取反） $$a\&(\sim a) = 0$$
---

三、按位或 $\mid$（二元运算）
#### 1. 交换律 $$a\mid b = b\mid a$$ 2. 结合律 $$(a\mid b)\mid c = a\mid(b\mid c)$$ 3. 幂等律 $$a\mid a = a$$ 4. 吸收律（搭配按位与） $$a\mid(a\&b) = a$$ 5. 分配律（对按位与分配） $$a\mid(b\&c) = (a\mid b)\&(a\mid c)$$ 6. 零元 / 单位元性质 $$a\mid 0 = a,\quad a\mid\text{全1掩码}=\text{全1掩码}$$ 7. 互补律（搭配按位取反） $$a\mid(\sim a) = \text{全1掩码}$$
---
四、按位异或 $\hat{}$（代码符号 `^`，二元运算）-> 重要结论：异或不满足对 $\&$、$\mid$ 的分配律
#### 1. 交换律 $$a\hat{}b = b\hat{}a$$ 2. 结合律 $$(a\hat{}b)\hat{}c = a\hat{}(b\hat{}c)$$ 3. 归零律（自身异或结果为0） $$a\hat{}a = 0$$ 4. 恒等律（异或0数值不变） $$a\hat{}0 = a$$ 5. 自逆可逆性质 若 $x = a\hat{}b$，则 $a = x\hat{}b,\ b = x\hat{}a$
