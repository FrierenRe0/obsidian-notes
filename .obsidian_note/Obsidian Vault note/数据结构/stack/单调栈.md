**用来解决在一个数组中，去找对于`nums[i]`在其右侧的（最近/最远）的一个更大的元素**
## 1.求最近的一个最大/最小元素：

**有两种思考方式**：
![配图](../../../截图文件/Pasted%20image%2020260423172327.png)

## 一、从右往左求「下一个更大/更小元素」（最经典）

这是标准的 “下一个更大元素” 模板，**时间复杂度 O (n)**。
```c++
// 输入 nums，返回每个元素右侧第一个比它大的元素，没有则返回 -1
vector<int> nextGreaterElement(vector<int>& nums) {
    int n = nums.size();
    vector<int> ans(n, -1);
    stack<int> st; // 存元素值（也可以存下标，看需求）

    // 从右往左遍历
    for (int i = n - 1; i >= 0; --i) {
        // 弹出栈顶比当前元素小的（它们不可能是任何左边元素的答案）
        while (!st.empty() && nums[st.top()] <= nums[i]) {
            st.pop();
        }
        // 栈顶就是第一个比 nums[i] 大的元素
        if (!st.empty()) {
            ans[i] = nums[st.top()];
        }
        // 当前元素入栈
        st.push(i);
    }
    return ans;
}
```

这是标准的 “下一个更小元素” 模板，**时间复杂度 O (n)**。

## 二、从左往右求「下一个更大元素」

和上面功能完全一样，只是遍历方向反过来，适合某些特定场景。

这是标准的 “下一个更大元素” 模板，**时间复杂度 O (n)**。
```c++
// 输入 nums，返回每个元素右侧第一个比它大的元素，没有则返回 -1
vector<int> nextGreaterElement(vector<int>& nums) {
    int n = nums.size();
    vector<int> ans(n, -1);
    stack<int> st; // 存下标！（核心区别）

    // 从左往右遍历
    for (int i = 0; i < n; ++i) {
        // 遇到更大的数，结算栈里的元素
        while (!st.empty() && nums[st.top()] < nums[i]) {
            ans[st.top()] = nums[i];
            st.pop();
        }
        st.push(i); // 压入下标
    }
    return ans;
}
```

这是标准的 “下一个更小元素” 模板，**时间复杂度 O (n)**。

## 2.找到最远的一个更大的元素
![配图](../../../截图文件/Pasted%20image%2020260423211220.png)

**代码思路**：
```c++
// 输入 nums，返回每个元素右侧最远的比它大的元素的下标，没有则返回 -1
int farthestGreaterElement(vector<int>& nums) {
    int n = nums.size();
    int ans = 0;
    stack<int> st; // 存下标，且栈内元素的 nums 值是单调递减的

    // 第一步：构建单调递减栈（只存可能成为“最远”答案的下标）
    for (int i = 0; i < n; ++i) {
        if (st.empty() || nums[i] < nums[st.top()]) {
            st.push(i);
        }
    }

    // 第二步：从右往左遍历，找每个栈顶元素能匹配的最远的更大值
    for (int i = n - 1; i >= 0; --i) {
        while (!st.empty() && nums[i] > nums[st.top()]) {
            ans = max(ans,i - st.top());
            st.pop();
        }
        if(st.empty()) break;
        if(i == st.top()) st.pop();
    }
    return ans;
}
```
