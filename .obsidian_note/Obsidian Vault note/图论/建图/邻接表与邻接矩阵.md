邻接表通常是来处理稀疏图，邻接矩阵通常处理稠密图；

## 1.稀疏表的建立

```c++
1.对于邻接表的板子：(含边权)

vector<vector<pair<int, int>>> graph(n + 1);  // 节点编号1~n，所以大小为n+1

for (int i = 0; i < m; ++i) {

    int u, v, w;// u,v 是两个节点，w是两个节点的边权

    cin >> u >> v >> w;

    //下面写法是建一个无向边

    //如果是有向边根据方向写一个就行

    graph[u].emplace_back(v, w);  // 无向图：u→v，权重w

    graph[v].emplace_back(u, w);  // 无向图：v→u，权重w

}
